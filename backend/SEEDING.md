# Seeding

## Quick start
```bash
cd backend
npm run seed
# or destructive reset:
npm run seed:reset
```

## Defaults
- Email: **admin@example.com**
- Password: **Admin@123**
- Username: **superadmin**

Override via env:
```
SEED_ADMIN_EMAIL=your@email
SEED_ADMIN_PASS=yourPassword
SEED_ADMIN_USERNAME=yourName
```
